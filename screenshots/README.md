# Screenshots

Esta carpeta contiene capturas de pantalla de la aplicación OpenVPN Manager CLI.

## Cómo Añadir Screenshots

1. Toma capturas de pantalla de la aplicación en uso
2. Guárdalas en esta carpeta con nombres descriptivos
3. Actualiza este README con las descripciones

## Screenshots Recomendados

Para mostrar la funcionalidad completa de la aplicación, considera añadir:

### 1. Menu Principal (`01-menu-principal.png`)
Captura del menú principal mostrando todas las opciones disponibles.

### 2. Conexiones Activas (`02-conexiones-activas.png`)
Vista de la tabla de usuarios conectados con estadísticas de tráfico.

### 3. Logs del Servidor (`03-logs-servidor.png`)
Visualización de logs con colores según el tipo de mensaje.

### 4. Revocar Usuario (`04-revocar-usuario.png`)
Proceso de revocación de certificado con lista de usuarios.

### 5. Estadísticas (`05-estadisticas.png`)
Panel de estadísticas generales y top usuarios por tráfico.

### 6. Configuración (`06-configuracion.png`)
Vista de la configuración actual del sistema.

## Formato

- **Formato preferido**: PNG
- **Resolución mínima**: 1280x720
- **Fondo**: Terminal con fondo oscuro preferiblemente

## Ejemplo de Uso en README

Una vez que añadas las screenshots, actualiza el README principal:

```markdown
## Capturas de Pantalla

### Menú Principal
![Menu Principal](screenshots/01-menu-principal.png)

### Conexiones Activas
![Conexiones Activas](screenshots/02-conexiones-activas.png)

### Estadísticas
![Estadísticas](screenshots/05-estadisticas.png)
```

## Tips para Buenas Screenshots

1. **Usa datos de ejemplo**: No muestres información sensible real
2. **Terminal de tamaño decente**: No muy pequeña ni muy grande
3. **Colores**: Asegúrate de que los colores se vean bien
4. **Contexto**: Muestra suficiente información para entender la funcionalidad
5. **Limpieza**: Cierra otras ventanas, limpia el escritorio

## Herramientas Recomendadas

- **Linux**: `gnome-screenshot`, `scrot`, `flameshot`
- **macOS**: `Cmd+Shift+4`
- **Captura de terminal**: `asciinema` para GIFs animados
